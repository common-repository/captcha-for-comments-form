== Description ==
This is a very basic plugin but work efficiently. Any suggestions are welcomed and I assure users that I will make
changes if it's in favor of the plugin. This plugin is using google recaptcha and reason for making this plugin is the
same as others, I was facing spam comments too.

== Installation ==
1. Upload Comments form captcha in WP comments form plugin to your wordpress `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. After the activation, have your Google API Keys pair ready or follow the instruction from https://developers.google.com/recaptcha/intro.
4. Go to `Settings > Comments Form Captcha`, add Google API Keys and save it. That's all.
